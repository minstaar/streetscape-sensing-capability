# 스트리트뷰 이미지의 상권 분석 능력과 한계
### — 선택적 센서로서의 정량적 능력지도 —

서울 발달·골목 상권 **825개**를 대상으로, GSV(POI 수집) 이미지가 상권의 무엇을 감지할 수 있고
없는지를 정량 매핑한 연구. 전체 보고서: **`상권_스트리트뷰_능력지도_연구보고서.docx`**.

> 한 줄 요약: 스트리트뷰는 상권의 **상업 규모·등급**(직장인구·점포·유형)을 강하게 감지하는 *선택적 센서*이며,
> 그 예측력은 **해석가능 구성비(44%) + 저수준 시각통계(~28%) + 거대비전모델 고유신호(~28%)** 로 분해된다.
> 주거·흐름·미시 경관 품질의 독립적 매출 효과는 구조 통제 후 **경계(association)** 에 머문다.

---

## 연구 질문
- **RQ1 (능력지도)** 이미지 단독으로 상권의 어떤 구조변수를 복원하는가?
- **RQ2 (메커니즘)** 그 능력은 어떤 시각요소에서 오며, 블랙박스 신호는 얼마나 해석가능한가?
- **RQ3 (경계)** 구조변수를 통제한 뒤에도 이미지가 매출을 추가로 설명하는가?

## 핵심 결과

**RQ1 — 능력지도 (이미지 단독, 행정데이터 0개)**

| 강 (규모·등급) | 약 | 미미 |
|---|---|---|
| 직장인구 R²=0.55 · 점포수 0.48 · 유형 AUC 0.94 | 상주 0.25 · 식음료 0.23 · 유동 0.17 · 폐업 0.11 | 개업 0.03 |

→ 가장 '번화함'을 대리하는 **유동인구를 거의 못 맞힘**(0.17) = "큰 상권 통째 암기"가 아님.
유형 내 분해 시 선택성은 주로 between-type이며, 발달상권 내부에선 직장인구 선택성이 유지됨.

→ *브리지*: 이미지는 **매출도 예측**(log매출 0.51·매출/유동 0.54)하지만, RQ3에서 인구·유형을 통제하면
추가 기여가 작아져 — 즉 이미지는 *대체로 구조(직장인구·규모)의 대리지표*임이 드러남.

**RQ2 — 메커니즘 + 블랙박스 해부**
- 반직관 녹지: 녹지는 골목(0.027)보다 **발달상권(0.047)에서 2배**, 직장인구와 +0.33 상관 → "가로수·조경 오피스 대로"의 지표.
- Commonality 분해(직장인구): **공유 44% + 텍스처 ~28% + DINO 고유 ~28%**. 핸드크래프트 텍스처가
  '환원 불가' 신호의 절반을 흡수(규명) + 약 28%는 거대비전모델만 포착(방어).

**RQ3 — 경계 (null)**
- 인구·면적·자치구·유형 통제 후 풀링 ΔR²=+0.05 (p<0.01)이나, 이는 **유형·잔여 규모 대리(association)**.
  골목 +0.104(p<0.001)는 인과 검증이 필요한 미해결 단서. **경관→매출 인과 주장 아님.**
